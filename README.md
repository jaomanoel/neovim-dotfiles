# neovim-dotfiles
## Custom Keys

### Move Lines

| Command | Description   |
|---------|---------------|
| Alt - s | Move down     |
| Alt - w | Move up       |

### Move to Window

| Command | Description           |
|----------|----------------------|
| Ctrl - a | Move to left window  |
| Ctrl - s | Move to lower window |
| Ctrl - w | Move to upper window |
| Ctrl - d | Move to right window |

### Buffers

| Command   | Description          |
|-----------|----------------------|
| Shift - a | Prev buffer          |
| Shift - d | Next buffer          |

### Terminal

| Command   | Description             |
|-----------|-------------------------|
| F7        | Open floating terminal  |
| F7        | Close floating terminal |
